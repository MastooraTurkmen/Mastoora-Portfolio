Personal Dashboard

A Chrome Extension that gives you an overview over the current time, how the weather is like and how Dogecoin is performing. Interacts with multiple APIs using async JS.

Live Link

Project Overview

Personal Dashboard is a powerful Chrome Extension that provides users with a convenient and customizable overview of essential information right at their fingertips. This feature-rich extension combines the functionalities of displaying the current time, weather updates, and real-time performance data for Dogecoin, all in one place. Developed using HTML, CSS, and JavaScript, Personal Dashboard offers a visually appealing and user-friendly interface. The extension seamlessly integrates with various APIs to gather and display up-to-date information. Leveraging async JavaScript, the extension fetches data from multiple sources asynchronously, ensuring a smooth and responsive user experience.

With Personal Dashboard, users can effortlessly stay informed about the current time, eliminating the need for manual checks or separate time-tracking tools. The extension also provides real-time weather updates, allowing users to plan their day accordingly and stay prepared for changing weather conditions.
