Password Generator

An app that generates random passwords based on a few user inputs, e.g. length, types of characters, etc. Built with HTML, CSS, and vanilla JavaScript.

Live Link

Project Overview

Password Generator is a practical and secure application designed to generate strong and random passwords based on user inputs. This user-friendly app allows users to customize the password generation process by specifying parameters such as the desired password length and types of characters to include. Built with HTML, CSS, and JavaScript, the Password Generator provides a seamless user interface that is both visually appealing and intuitive to navigate. Users can easily interact with the app, input their preferred password specifications, and generate a strong and unique password with just a few clicks.

The app's core functionality lies in its ability to generate passwords that meet the user's specific requirements. Users can define the length of the password, ranging from a minimum to a maximum value. Additionally, they can select the types of characters to include, such as uppercase letters, lowercase letters, numbers, and special symbols.
